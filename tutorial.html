How to Use the Adaptive Map Tool

Below is a detailed video tutorial explaining how to use the features and functions of the tool.

The user interface of the adaptive map tool organizes the content into three visual levels. To navigate around in the level, you can pan by clicking on the nodes (which centers the screen on the selected node) or by clicking in an empty space, holding the mouse button and dragging to move the screen around manually. To zoom in or out, use the scroll wheel on your mouse, or if you do not have a scroll wheel, use the “+” and “-“ buttons on the screen.

The three views are from broadest to most detailed are…

  1. Overview – Where you will get a visual overview of all the information in engineeering statics, simplified into the major clusters of ideas with visual connections. Each of the cluster nodes represents a number of highly related ideas, each with it's own page.
  2. Cluster View – If you zoom into one of the cluster nodes in the overview, you will transition to the cluster view. In this view, the single cluster node will expand into a number of idea nodes (the cluster node represents all of these nodes) and the relationships between the idea nodes will appear. Notice that all of the idea nodes match the cluster and the background color shifts to match the cluster you are in. If you click on one of these idea nodes you will focus on it. This will center the view on the selected idea node and will add any idea nodes from other clusters that are directly related to the idea selected. If you click on one of these external idea nodes you will shift to another cluster.
  3. Page View – Double clicking on any of the idea nodes will open a webpage on the selected idea. This is the page view. In the page view you will find text and images explaining the idea and you will usually find worked problems.
